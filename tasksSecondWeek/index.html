Post 2: What is IPFS?

The InterPlanetary File System (IPFS) is a decentralized network for storing and sharing files. It enables a fully distributed web where nodes communicate directly via a peer-to-peer (P2P) network.

Post 3: What is ENS?

Ethereum Naming Service (ENS) is a decentralized naming system that links human-readable names to Ethereum addresses.
